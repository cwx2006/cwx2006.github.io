如何优雅隐藏 Hexo 文章 | CC的部落格

Hexo NexT 置顶

  • 卸载默认的 hexo-generator-index 插件：

npm uninstall hexo-generator-index –save

  • 安装 hexo-generator-index-pin-top 插件：

npm install hexo-generator-index-pin-top –save

  • 在需要置顶的文章的 Front-matter 中添加 top 属性。例如：

-–

title: “示例文章”

date: 2023-09-01

top: 2

-–
